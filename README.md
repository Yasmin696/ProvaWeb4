# 📚 Clubinho do Livro

## 📖 Sobre o projeto

O **Clubinho do Livro** é um site desenvolvido para apresentar um clube de assinatura literário. O objetivo é incentivar a leitura por meio de caixas mensais com livros selecionados, além de permitir que os usuários conheçam as obras mais populares e indiquem novos títulos.

Este projeto foi desenvolvido para fins acadêmicos na disciplina de Desenvolvimento Web.

---

## ✨ Funcionalidades

* Página inicial com apresentação do clube.
* Navegação por menu responsivo.
* Exibição dos livros favoritos do clube.
* Cards contendo:

  * Capa do livro;
  * Título;
  * Autor;
  * Avaliação;
  * Descrição;
  * Botão de interesse.
* Seção de avaliações dos leitores.
* Formulário para indicação de novos livros.
* Layout moderno e responsivo.
